[LeetCode] 226. Invert Binary Tree

問題描述

Invert a binary tree.

Example

翻譯

翻轉這顆二元樹

解題思維

  1. 使用 level Order 方式拜訪這顆二元樹，用佇列將每一層  節點儲存起來
  2. 同時檢查拜訪該節點是否有左右子樹，若有加入佇列裡面
  3. 交換左右節點
  4. 重複第二、三步驟直至佇列為空
